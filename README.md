# Customer Churn Prediction

## Описание проекта

В этом проекте решается задача предсказания оттока клиентов (churn).
Модель определяет вероятность того, что клиент уйдёт.

Датасет содержит информацию о клиентах телеком компании:

* тип контракта
* длительность обслуживания
* ежемесячные платежи
* дополнительные услуги

---

## Цель

Построить модель, которая:

* предсказывает вероятность оттока
* помогает находить клиентов с высоким риском ухода

---

## Используемые технологии

* Python
* pandas
* numpy
* scikit-learn

---

## Этапы работы

### 1. Анализ данных

* проверка типов данных
* поиск пропусков
* анализ распределений
* изучение связи признаков с churn

---

### 2. Предобработка данных

* преобразование TotalCharges в числовой формат
* удаление пропусков
* удаление неинформативных признаков (customerID)
* кодирование целевой переменной (Yes/No → 1/0)

---

### 3. Разделение данных

* train/test split (80/20)
* стратификация по целевой переменной

---

### 4. Подготовка признаков

* числовые признаки масштабируются (StandardScaler)
* категориальные кодируются (OneHotEncoder)

---

### 5. Модель

Используется логистическая регрессия:

* max_iter = 1000
* class_weight = balanced

---

### 6. Pipeline

Весь процесс объединён в Pipeline:

* preprocessing
* обучение модели

Это позволяет:

* избежать утечки данных
* упростить обучение и предсказание

---

### 7. Предсказание

Модель возвращает вероятность оттока:

```python
y_proba = pipeline.predict_proba(X_test)[:, 1]
```

---

### 8. Метрики

Используются:

* ROC-AUC
* F1-score

---

### 9. Подбор порога

Порог классификации подбирается вручную:

```python
thresholds = np.linspace(0.1, 0.6, 10)
```

Это позволяет улучшить качество модели с точки зрения F1.

---

## Результат

* модель умеет оценивать вероятность оттока
* найден оптимальный threshold
* можно выделять клиентов с высоким риском

---

## Как использовать

1. Обучить модель:

```python
pipeline.fit(X_train, y_train)
```

2. Получить вероятности:

```python
y_proba = pipeline.predict_proba(X_test)[:, 1]
```

3. Применить threshold:

```python
y_pred = (y_proba > 0.6).astype(int)
```
